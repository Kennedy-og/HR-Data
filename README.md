# HR Attrition Analysis 

This project features a Tableau visualization designed to explore employee attrition across
age groups, genders, job roles, departments, and education fields. The dashboard provides a clear, interactive overview of workforce dynamics and supports data-driven storytelling.

## Overview

The HR Attrition Dashboard helps identify patterns and insights behind employee turnover. 
By analyzing variables such as age, gender, satisfaction ratings, and education fields, 
the dashboard provides a broad and interactive perspective on how workforce characteristics influence attrition.

---

## Project Objectives

- Visualize employee attrition using an interactive Tableau dashboard.  
- Investigate attrition across age groups, gender, roles, and departments.  
- Apply effective storytelling and layout techniques in Tableau.  
- Improve analytical communication using visual insights.

## Dashboard Highlights

- Attrition breakdown by **gender**, **age**, and **department**  
- Job satisfaction levels across multiple job roles  
- Education fields with the highest attrition  
- Donut charts reflecting gender-age attrition distribution  
- Histogram showing employee distribution by age  
- Clean top summary panel for quick insights
